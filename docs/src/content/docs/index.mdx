## Why Lume?

<CardGrid stagger>
  <Card title="~1 kB Gzipped" icon="rocket">
    Incredibly small. Zero dependencies. Ships nothing you don't use.
  </Card>
  <Card title="Signal-Based Reactivity" icon="star">
    Fine-grained reactive updates via signals and effects — no dirty checking, no VDOM diffing.
  </Card>
  <Card title="HTML-First" icon="document">
    Attach logic to existing HTML. Perfect for server-rendered pages, CMS sites, or progressive enhancement.
  </Card>
  <Card title="TypeScript Native" icon="approve-check">
    Written in TypeScript with full type inference. No `@types` packages needed.
  </Card>
